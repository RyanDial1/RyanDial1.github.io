Covid Statistics on a Global and Country Level
  • GitHub

Description

I worked on this weekend project because I had a growing interest in what different API's can accomplish and wanted to also practice my HTML, CSS, and JavaScript skills. This JavaScript project fetches data from the Covid19API. The documentation for this API is extensive, ranging from stats for a specific country or on a global scale. Here we will display data including total cases, total deaths, and new deaths globally and for the US.

Procedure

The data is displayed in a simple table format. Clicking the button will update the data. The data has been collected since the pandemic began in March 2020.

Console

Here is where the information in stored by logging the data from the covid19API to the console.

Code

Simple function that performs a GET request to the covid19API url. The summary section contains the specific information we need to display the total cases, deaths since the start of the pandemic and new deaths as of the day that the request was sent. The function uses AJAX syntax to fetch the data and populates it with the corresponding HTML tags.
