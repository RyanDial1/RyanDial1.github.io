Procedure

This is how a page from Newegg looks like, we want to extract certain information from it, such as the price, product name, and page numbers (since the program will need to find the cheapest price among all products).

Code

We start off by pasting the exact link used when searching for a product. Then we input this url into the BeautifulSoup module that will parse the html so we can look for specific tags. Next, we get the number of pages that a product has. By using Google Chrome's inspect function, we discover that the number of pages can be found under the class name "list-tool-pagination-text". Once we extract the number of pages, we also find the specific product's parent tag by using the find or find_all function. Once these tags were found, all that was left was to sort them and print them out in a list that will be organized in descending order, which shows the cheapest products at the top of the list!
