Description

This was probably the most fun recent project I've worked on. I followed a tutorial (credits to TechWithTim) that finds the cheapest computer products on Newegg, the online shopping store. My original goal was to find the cheapest 3080 graphics cards on the market, but this program also works for other products as well! But this is pretty much similar to any website scraper out there, I just wanted to implement something useful while practicing my python skills at the same time. A Website Scraper essentially parses an HTML file and extracts certain information from it. In this case, it will gather a collection of the cheapest products and provide the price and the link!

Procedure

This is how a page from Newegg looks like, we want to extract certain information from it, such as the price, product name, and page numbers (since the program will need to find the cheapest price among all products).

HTML tags that contain the information we want

For example, we can see that the name of this specific product is located within HTML tags with a specific name. Beautiful Soup allows us to search for those tags.

Code

We start off by pasting the exact link used when searching for a product. Then we input this url into the BeautifulSoup module that will parse the html so we can look for specific tags. Next, we get the number of pages that a product has. By using Google Chrome's inspect function, we discover that the number of pages can be found under the class name "list-tool-pagination-text". Once we extract the number of pages, we also find the specific product's parent tag by using the find or find_all function. Once these tags were found, all that was left was to sort them and print them out in a list that will be organized in descending order, which shows the cheapest products at the top of the list!

Searching for Cheapest 3080 graphics card

At this point in time, we can see that the cheapest 3080 on Newegg is still over $2,000!
